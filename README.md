<h2>ピーマンの特徴</h2>

<p>
ピーマンはナス科トウガラシ属の野菜で、独特の苦みと香りが特徴です。
ビタミンCやβカロテンなどの栄養素を豊富に含んでおり、加熱しても栄養が失われにくい野菜です。
炒め物やサラダ、肉詰めなどさまざまな料理に利用され、日本でも広く栽培されています。
比較的育てやすく、家庭菜園でも人気があります。
</p>

<h2>栽培のポイント</h2>

<ul>
    <li>日当たりと風通しのよい場所で育てる。</li>
    <li>土が乾いたら十分に水やりをする。</li>
    <li>追肥を定期的に行い、栄養不足を防ぐ。</li>
    <li>実を長く収穫するために適度に剪定する。</li>
</ul>
